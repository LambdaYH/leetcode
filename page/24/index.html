递归 Brute Force

枚举出所有情况，然后对每种情况判断是否是合法的括号对，用到了20. Valid Parentheses中的思路

BackTracking

仅仅当括号符合添加的条件时才添加这个括号。
当左括号小于N的时候，表示可以加左边括号，那就加，当左括号大于右边括号时，那就加上右边括号来闭合。

Closure Number

第一个 n == 0 时添加一个""是为了让他可以进入一次循环计算右边项而不是直接return
c = (2c + 1 - 0 - 1）/2
n - c - 1 = (2n - 2c - 1 - 1)/2
